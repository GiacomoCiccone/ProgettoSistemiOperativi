Per implementare le funzioni richieste abbiamo deciso di creare 2 file di header (uno per pcb, uno per asl)
in cui sono riportate le definizioni delle funzioni con descrizione del loro significato
e 2 file c in cui le funzioni sono implementate.
Tutte le funzioni sono commentate, spesso anche riga per riga,
quindi rimandiamo al codice per la spiegazione delle nostre implementazioni.

Per compilare il progetto bisogna utilizzare il comando make.
